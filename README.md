## 🚀 Featured Projects

### 🔹 [FinRisk Investigator](https://github.com/nandini06/FinRisk)

An explainable financial risk investigation backend that combines structured transaction data, semantic retrieval, anomaly scoring, and LLM-based report generation.

**Highlights:**
- Built an agentic RAG workflow for financial risk analysis
- Used PostgreSQL for structured transaction and customer data
- Used ChromaDB for semantic retrieval of policies and historical cases
- Integrated Gemini API for planning, reporting, and verification
- Added self-corrective retrieval when evidence is insufficient

**Tech Stack:** Python, FastAPI, PostgreSQL, ChromaDB, Gemini API, RAG

---

### 🔹 [Scam Detection](https://github.com/nandini06/scam_detection)

A Python-based scam detection project focused on processing conversation data, preparing chunks, building embeddings, and testing retrieval workflows.

**Highlights:**
- Built data ingestion and preprocessing scripts
- Created chunking and embedding pipeline
- Built FAISS-based retrieval flow
- Designed RAG-style prompts for scam detection analysis

**Tech Stack:** Python, FAISS, NLP, RAG
